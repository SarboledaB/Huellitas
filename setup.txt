**PHP version**
PHP 7.4.15
**Composer version**
2.0.9

**package installation**
Go to the terminal (you should be placed on the project root directory). Then, run the following commands:
    $ composer install

**rename /.env.example into /.env**
Go to the terminal (you should be placed on the project root directory). Then, run the following commands:
    $ mv .env.example .env
or do it manually


**Connect to PhpMyAdmin and create a new database (called tallerLaravel)**


**Modify the /.env file with the corresponding database connection values:**
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=tallerLaravel
DB_USERNAME=user
DB_PASSWORD=yourpassword



**Make migrations**
Go to the terminal (you should be placed on the project root directory). Then, run the following commands:
    $ php artisan migrate
It should generate the products table (and other Laravel tables). Examine them in phpMyAdmin the generated table.
You should get something like this:
    Migration table created successfully.
    Migrating: 2014_10_12_100000_create_password_resets_table
    Migrated:  2014_10_12_100000_create_password_resets_table (39.40ms)
    Migrating: 2019_08_19_000000_create_failed_jobs_table
    Migrated:  2019_08_19_000000_create_failed_jobs_table (33.45ms)
    Migrating: 2021_02_28_000001_create_users_table
    Migrated:  2021_02_28_000001_create_users_table (18.72ms)


**Make seed**
Go to the terminal (you should be placed on the project root directory). Then, run the following commands:
    $ php artisan db:seed
It should generate the product “fake” rows (go to your product table and check it)
You should get something like this:
    Database seeding completed successfully.


**Run project**
    $ php artisan serve

""Go to Home page (Activity 1)**
http://127.0.0.1:8000/

**Go to Activity 2 clicking on "Create User"**

**Try to create a new user (Activity 3)**
If you do it properly, you should get a message like: "User created successfully!"

**Go to Activity 4 and 7 clicking on "List Users"**
You should be able to see all Users sort by Id
And the first and second one's id should be in bold

**Go to Activity 5 clicking on any user id**
You should be able to see the user's information

**Go to Activity 6 clicking on "Delete"**
If you do it properly, you should get a message like: "User deleted successfully!"
