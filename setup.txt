**PHP version**
PHP 7.4.15 
**Composer version**
2.0.9

**package installation**
$ composer install

**Connect to MySQL**
$ mysql -u root -p

**Create a new use**
CREATE USER 'user'@'localhost' IDENTIFIED WITH mysql_native_password BY 'yourpassword';

**Create a new database**
CREATE DATABASE laravelcourse;

**Grant all privileges to that user to the laravelcourse database**
GRANT ALL PRIVILEGES ON laravelcourse.* TO 'user'@'localhost' WITH GRANT OPTION;

**Modify the /.env file**
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravelcourse
DB_USERNAME=user
DB_PASSWORD=yourpassword

**Make migrations**
$ php artisan migrate

**Make seed**
$ php artisan db:seed

**Run project**
$ php artisan serve
